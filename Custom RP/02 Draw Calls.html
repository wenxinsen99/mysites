Draw Calls

SRP Batcher

批处理(Batching)是将多个 draw call 合并的过程，减少 CPU 和 GPU 之间的通信时间。

Unity 的 SRP Batcher 允许在不使用 GPU 实例化(Instancing) 的情况下批处理 draw call。SRP Batcher 通过将多个材质相同的物体合并为一个 draw call 来减少 draw call 的数量。

GPU Instancing

GPU 实例化(Instancing)是将多个相同的物体合并为一个 draw call 的过程。GPU 实例化允许在不使用 SRP Batcher 的情况下批处理 draw call。

GPU 实例化通过将多个相同的物体合并为一个 draw call 来减少 draw call 的数量。
